# 📖 [my_dashboard_nextjs] <a name="about-project"></a>

**[my_dashboard_nextjs]** is an intuitive dashboard created to manage shopping cart items, explore Pokémon details, and organize favorites. Enhanced with Redux Toolkit for state management, dynamic data fetching, and localStorage integration.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
  Visual Studio Code
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:Nelitaa/my-dashboard-nextjs.git
```

### Install

Install this project with:

```sh
  cd my-project
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```
